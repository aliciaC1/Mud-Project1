Müd : /myːt/

A web application that enables users to keep an emotional diary of their daily emotions through photos, tags and comments, to help users identify triggers behind certain emotions.

View Source Code on Github.

Concept

Functions:
  • Photo Log: keeps log of photos taken daily. Allows users to view variations in their emotions and facial appearances overtime.
  • Overview of emotion patterns over the course of a year.
  • Capability to tag daily emotions / daily photos with key words related to details that happened that day. Which then Allows uers to review /search images through specific tags.
  • Note taker: allows user to input comments or more detailed information about their day. Text is associated with photo, for more detailed records.
  • Search images / emotional details through different paramters such as dates, emotion cateogries, keywords, that will help user find patterns in emotional triggers
  • Users can download / share images taken through web application, individually or as time lapse.
  • Aid in healthcare, face mapping
  • Müd Board: Dashboard filled with supporting media that reflects their daily emotion. Space for user to write comments, create keyword tags, search for suggested places near their location, view/ listen to curated content provided based on their mood. Customizeable, art/ journaling space for users to express emotion through dragging / resizing content from default. user can save image of their daily dashboard, or share image through social media, instagram, twitter, facebook.
